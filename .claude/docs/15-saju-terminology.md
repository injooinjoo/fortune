## 1. 기본 개념 (Basic Concepts)

### 1.1 사주팔자(四柱八字)

**정의**: 사람이 태어난 년(年), 월(月), 일(日), 시(時)를 천간(天干)과 지지(地支)로 표현한 것

- **사주(四柱)**: 4개의 기둥 = 년주, 월주, 일주, 시주
- **팔자(八字)**: 8개의 글자 = 4개 천간 + 4개 지지

```
예시: 1988년 9월 5일 축시(01:00-03:00) 출생
┌────────┬────────┬────────┬────────┐
│  시주  │  일주  │  월주  │  년주  │
├────────┼────────┼────────┼────────┤
│   癸   │   辛   │   庚   │   丙   │ ← 천간
│   丑   │   卯   │   寅   │   子   │ ← 지지
└────────┴────────┴────────┴────────┘
```

### 1.4 오행(五行) - Five Elements

| 화(火) | 火 | #F44336 | #E57373 | 여름 | 남 | 열정, 예절, 예(禮) |
| 토(土) | 土 | #FF9800 | #FFB74D | 환절기 | 중앙 | 안정, 신뢰, 신(信) |
| 금(金) | 金 | #C0C0C0 | #E0E0E0 | 가을 | 서 | 결단, 의리, 의(義) |

**상생(相生) - 서로 돕는 관계**:
```
木 → 火 → 土 → 金 → 水 → 木
(나무가 불을 지피고, 불이 재(흙)가 되고, 흙이 쇠를 품고, 쇠가 물을 맺히게 하고, 물이 나무를 키운다)
```

**상극(相剋) - 서로 제어하는 관계**:
```
木 → 土 → 水 → 火 → 金 → 木
(나무가 흙을 뚫고, 흙이 물을 막고, 물이 불을 끄고, 불이 쇠를 녹이고, 쇠가 나무를 벤다)
```

### 2.1 십신(十神) - Ten Gods

일간(日干)을 기준으로 다른 천간/지지와의 관계를 나타내는 10가지 분류.

| 십신 | 한자 | 관계 | 의미 | 특성 |
|------|------|------|------|------|
| 비견 | 比肩 | 같은 오행, 같은 음양 | 형제, 동료 | 경쟁, 독립심, 자존심 |
| 겁재 | 劫財 | 같은 오행, 다른 음양 | 경쟁자 | 도전, 변화, 투쟁 |
| 식신 | 食神 | 내가 생(生)하는 오행, 같은 음양 | 재능 | 창조, 표현, 여유 |
| 상관 | 傷官 | 내가 생하는 오행, 다른 음양 | 예술성 | 반항, 자유, 창의 |
| 편재 | 偏財 | 내가 극(剋)하는 오행, 같은 음양 | 사업재 | 투자, 변동, 모험 |
| 정재 | 正財 | 내가 극하는 오행, 다른 음양 | 근로재 | 안정, 저축, 근면 |
| 편관 | 偏官 | 나를 극하는 오행, 같은 음양 | 권력 | 도전, 변혁, 직관 |
| 정관 | 正官 | 나를 극하는 오행, 다른 음양 | 명예 | 책임, 규율, 지위 |
| 편인 | 偏印 | 나를 생하는 오행, 같은 음양 | 직관 | 학문, 종교, 독창성 |
| 정인 | 正印 | 나를 생하는 오행, 다른 음양 | 학문 | 교육, 보호, 자격증 |

**계산 방법**:
```dart
String calculateTenshin(String ilGan, String targetGan) {
  final ilGanWuxing = getWuxing(ilGan);
  final targetWuxing = getWuxing(targetGan);
  final isSameYinYang = (tianGan.indexOf(ilGan) % 2) == (tianGan.indexOf(targetGan) % 2);

  if (ilGanWuxing == targetWuxing) {
    return isSameYinYang ? '비견' : '겁재';
  } else if (generates(ilGanWuxing, targetWuxing)) {
    return isSameYinYang ? '식신' : '상관';
  } else if (generates(targetWuxing, ilGanWuxing)) {
    return isSameYinYang ? '편인' : '정인';
  } else if (controls(ilGanWuxing, targetWuxing)) {
    return isSameYinYang ? '편재' : '정재';
  } else {
    return isSameYinYang ? '편관' : '정관';
  }
}
```

### 2.4 대운(大運) - Major Fortune

10년 주기로 변화하는 운의 흐름. 월주를 기준으로 계산합니다.

**대운 계산 원칙**:
- **양남음녀(陽男陰女)**: 년간이 양간인 남자, 년간이 음간인 여자 → 순행
- **음남양녀(陰男陽女)**: 년간이 음간인 남자, 년간이 양간인 여자 → 역행

**대운수 계산**:
```
대운수 = (생일 ~ 절기까지 일수) ÷ 3
```

### 3.7 공망(空亡) - Emptiness

60갑자 순환에서 천간 10개가 지지 12개를 모두 채우지 못해 2개가 빠지는 현상.
일주를 기준으로 계산하며, 공망에 해당하는 지지는 작용이 약해집니다.

| 순(旬) | 범위 | 공망 지지 |
|--------|------|-----------|
| 갑자순 | 갑자~계유 | 술(戌), 해(亥) |
| 갑술순 | 갑술~계미 | 신(申), 유(酉) |
| 갑신순 | 갑신~계사 | 오(午), 미(未) |
| 갑오순 | 갑오~계묘 | 진(辰), 사(巳) |
| 갑진순 | 갑진~계축 | 인(寅), 묘(卯) |
| 갑인순 | 갑인~계해 | 자(子), 축(丑) |

**계산 방법**:
```dart
List<String> calculateGongMang(String dayGan, String dayZhi) {
  final ganIndex = tianGan.indexOf(dayGan);
  final zhiIndex = diZhi.indexOf(dayZhi);

  // 순(旬) 시작점 계산
  final xunStartZhi = (zhiIndex - ganIndex + 12) % 12;

  // 공망은 순 시작점의 -2, -1 위치
  final gongMang1 = diZhi[(xunStartZhi + 10) % 12];
  final gongMang2 = diZhi[(xunStartZhi + 11) % 12];

  return [gongMang1, gongMang2];
}
```

## 4. 계산 알고리즘 (Calculation Algorithms)

### 4.1 년주(年柱) 계산

**원칙**: 입춘(立春, 양력 2월 4일경)을 기준으로 년이 바뀜

```dart
Map<String, String> calculateYearPillar(DateTime birthDate, bool isLunar) {
  int year = birthDate.year;

  // 입춘 전이면 전년도로 계산
  final lichun = getSolarTerm(year, '입춘'); // 그 해 입춘 날짜
  if (birthDate.isBefore(lichun)) {
    year -= 1;
  }

  // 천간: (년도 - 4) % 10
  final ganIndex = (year - 4) % 10;
  // 지지: (년도 - 4) % 12
  final zhiIndex = (year - 4) % 12;

  return {
    'gan': tianGan[ganIndex],
    'zhi': diZhi[zhiIndex],
  };
}
```

### 4.3 일주(日柱) 계산

**원칙**: 만세력 데이터 또는 공식 계산

```dart
Map<String, String> calculateDayPillar(DateTime birthDate) {
  // 기준일: 1900년 1월 1일 = 갑진일(甲辰日)
  final baseDate = DateTime(1900, 1, 1);
  final baseDayGanIndex = 0;  // 갑
  final baseDayZhiIndex = 4;  // 진

  final daysDiff = birthDate.difference(baseDate).inDays;

  final ganIndex = (baseDayGanIndex + daysDiff) % 10;
  final zhiIndex = (baseDayZhiIndex + daysDiff) % 12;

  return {
    'gan': tianGan[ganIndex],
    'zhi': diZhi[zhiIndex],
  };
}
```
