# 비즈니스 모델 가이드

> 최종 업데이트: 2026.02.08

## 개요

Fortune 앱의 수익화 모델을 정의한 문서입니다.
**2가지 모델**: 무료, 토큰 소비

---

## 1. 수익화 모델 분류

### 1.1 모델 종류

| 모델 | 설명 | 토큰 필요 | 운세 수 |
|------|------|----------|--------|
| 🆓 **무료** | 제한 없이 무료 이용 | 불필요 | 5개 |
| 🪙 **토큰** | 토큰 소비 | 필수 | 31개 |

### 1.2 핵심 원칙

- **무료 운세**: 토큰 없이 무제한 이용
- **토큰 운세**: 반드시 토큰 소비
- **구독**: 토큰 정기구매 (매월 자동 충전)

---

## 2. 운세별 모델 배정

### 2.1 무료 (🆓) - 5개

| 운세 | 칩 ID | 설명 |
|------|-------|------|
| 오늘의 나 | `daily` | 매일 무료 제공 |
| 포춘쿠키 | `fortuneCookie` | 간단한 메시지 |
| 숨쉬기 | `breathing` | 웰니스 기능 |
| 감사일기 | `gratitude` | 웰니스 기능 |
| 오운완 | `exercise` | 운동 추천 |

### 2.2 토큰 (🪙) - 31개

| 운세 | 칩 ID | 토큰 | 비고 |
|------|-------|------|------|
| 새해 인사이트 | `newYear` | 🪙 5개 | 연간 운세 |
| 사주 분석 | `traditional` | 🪙 5개 | 전통 사주팔자 |
| 작명 | `naming` | 🪙 7개 | 이름 분석 |
| 태명 분석 | `babyNickname` | 🪙 5개 | 아기 태명 |
| 올해의 인연 | `yearlyEncounter` | 🪙 3개 | 이미지 생성 포함 |
| 유명인 궁합 | `celebrity` | 🪙 5개 | 셀럽 매칭 |
| 로또 번호 | `lotto` | 🪙 3개 | 번호 추천 |
| 시험운 | `exam` | 🪙 3개 | 시험 분석 |
| OOTD 평가 | `ootdEvaluation` | 🪙 3개 | 패션 이미지 생성 |

**토큰 운세 규칙:**
- ⚠️ 구독자도 토큰 소비 **필수**

## 3. 토큰 경제 시스템

### 3.1 획득 방법

| 방법 | 토큰 | 빈도 | 비고 |
|------|------|------|------|
| 출석 체크 | 🪙 2개 | 매일 | 기본 지급 |
| 연속 7일 출석 | 🪙 5개 | 주 1회 | 보너스 |
| 운세 공유 | 🪙 1개 | 일 3회 | 바이럴 마케팅 |
| 친구 초대 | 🪙 5개 | 초대당 | 1회성 |
| 리뷰 작성 | 🪙 10개 | 1회 | 1회성 |

**일일 최대 무료 획득**: 🪙 9개

## 5. 구독 (토큰 정기구매)

### 5.1 구독 상품 (iOS Tier 기준)

| 상품 | 티어 | 가격 | 토큰 | 할인율 |
|------|------|------|------|--------|
| 월간 구독 | Tier 1 | **₩1,500/월** | 🪙 50개/월 | - |
| 연간 구독 | Tier 12 | **₩15,000/년** | 🪙 600개/년 | 17% |

### 5.2 구독 혜택

| 혜택 | 설명 |
|------|------|
| 🪙 **월간 토큰 충전** | 매월 50개 자동 충전 |

**핵심**: 구독 = 토큰 정기구매. 구독자도 운세 이용 시 토큰 소비 필수.
