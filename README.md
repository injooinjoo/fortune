# 행운 (Fortune Compass)

**모든 운명은 당신의 선택에 달려있습니다.**

`행운`은 전통적인 지혜와 최신 AI 기술을 결합하여 사용자에게 깊이 있는 개인 맞춤형 운세 경험을 제공하는 풀스택 애플리케이션입니다. Google Genkit을 활용한 AI 분석을 통해, 당신의 삶에 대한 통찰력을 얻고 미래를 탐험하는 나침반이 되어 드립니다.

## 🏛️ 웹 애플리케이션 아키텍처

`행운`은 사용자가 방대한 운세 콘텐츠를 쉽고 직관적으로 탐색할 수 있도록 사용자 중심의 아키텍처로 설계되었습니다.

### 1. 사용자 중심 정보 구조 (IA)

기능들을 사용자 의도에 따라 재분류하여, 누구나 원하는 정보를 쉽게 찾을 수 있도록 구성했습니다.

- **핵심 운세 서비스:** 매일 확인하는 운세부터 심층 분석, 사용자가 직접 참여하는 인터랙티브 운세까지 운세의 핵심 기능을 그룹화했습니다.
- **특별 콘텐츠:** 연애, 취업 등 특정 상황에 대한 운세와 연예인 궁합 같은 흥미 위주 콘텐츠를 분리하여 제공합니다.
- **나의 운세 기록:** 과거 운세 기록과 통계 분석을 통해 자신의 운세 흐름을 추적하고 관리하는 개인화된 공간입니다.

### 2. 최적의 온보딩 경험

신규 사용자가 서비스의 가치를 빠르게 느끼고 이탈하지 않도록, `최소한의 노력으로 최대의 가치를 경험`하는 4단계 온보딩 프로세스를 설계했습니다.

1.  **간편 로그인:** 소셜 로그인으로 진입 장벽 최소화
2.  **핵심 정보 입력:** 정확한 운세 분석을 위한 필수 정보(생년월일) 입력
3.  **추가 정보 입력 (선택):** 더 깊은 개인화를 위한 MBTI, 출생 시간 등 선택적 정보 입력
4.  **즉각적인 가치 제공:** 정보 입력 즉시 **오늘의 총운**을 제공하여 서비스의 효용성을 바로 체감

### 3. 개인화 대시보드

로그인 후 가장 먼저 마주하는 메인 페이지는 위젯 기반의 개인화 허브로 작동합니다.

- **오늘의 운세 요약:** 가장 중요한 오늘의 운세를 상단에 고정 노출합니다.
- **맞춤 운세 피드:** MBTI, 별자리 등 개인화된 운세를 카드 형태로 제공합니다.
- **빠른 실행 도구:** 타로카드, 관상 분석 등 자주 사용하는 인터랙티브 기능에 빠르게 접근합니다.

### 4. 직관적인 네비게이션 시스템

모바일 웹 환경에 최적화된 **하단 탭 바 (Bottom Tab Bar)**를 통해 앱의 핵심 기능에 언제나 쉽게 접근할 수 있습니다.

- **🏠 홈:** 개인화된 대시보드
- **🧭 전체 운세:** 앱의 모든 운세 서비스를 체계적으로 탐색하는 라이브러리
- **✨ 스페셜:** 흥미 위주, 주제별 운세 등 특별 콘텐츠
- **📊 나의 기록:** 과거 운세 기록과 통계 분석
- **👤 프로필:** 계정 정보 및 설정

## 🌟 주요 기능 (Key Features)

사용자 중심의 정보 구조에 따라 재구성된 `행운`의 주요 기능입니다.

### 🔮 핵심 운세 서비스

#### 매일의 운세 (Daily Fortune)
- 오늘의 총운
- MBTI 유형별 일일/주간/월간 운세
- 띠별/별자리별 운세

#### 심층 분석 (In-depth Analysis)
- 사주팔자 상세 분석
- 토정비결 (연간 운세)
- 주역점 (상황별 운세)
- 풍수지리 (거주지, 사무실 분석)

#### 인터랙티브 운세 (Interactive Tools)
- 타로카드 (AI 기반 해석)
- 관상 분석 (AI 얼굴 분석)
- 손금 분석
- 꿈해몽 (키워드, 상황별 해석)

### ✨ 특별 콘텐츠

- **주제별 운세:** 연애/결혼/이별, 취업/시험/승진, 재물/금전 운세, 로또 번호 추천
- **흥미 위주 운세:** 연예인과의 궁합 분석, 이름풀이 및 작명, SNS 닉네임 운세, 반려동물 사주

### 📊 나의 운세 기록

- **히스토리:** 모든 운세 결과를 날짜별로 저장하고 과거 운세와 비교
- **통계 분석:** 운세 결과를 차트와 그래프로 시각화하여 운의 흐름 추적

### 👤 사용자 중심 기능

- **프로필 시스템:** 이름, 생년월일, MBTI, 성별, 출생 시간 등 개인화 정보 관리
- **소셜 로그인:** Google, 카카오를 통한 간편 인증
- **소셜 공유:** 운세 결과를 Instagram, Facebook 등 소셜 미디어에 공유
- **푸시 알림 및 위젯:** 일일 운세 알림, 홈스크린 위젯 지원

## 📋 페이지 리스트 (Page List)

### 1. 온보딩 및 인증 (Onboarding & Auth)

#### 1-1. 랜딩 페이지 (로그인 전)
**핵심 목표:** 서비스의 핵심 가치(AI 개인 맞춤 운세)를 매력적으로 전달하고, 사용자의 로그인 및 회원가입을 유도.

**주요 구성 요소:**
- **헤더:** 서비스 로고, 로그인 버튼
- **히어로 섹션:** "AI가 분석하는 당신만의 운명"과 같은 매력적인 헤드라인, 서비스의 특징을 보여주는 동적인 그래픽 또는 배경 영상
- **CTA (Call to Action) 버튼:** Google로 시작하기, 카카오로 시작하기 등 소셜 로그인 버튼을 가장 눈에 띄게 배치
- **기능 소개 섹션:** '사주팔자', 'AI 관상', '타로카드' 등 핵심 기능을 아이콘과 짧은 설명으로 소개

**사용자 인터랙션:** 스크롤을 통해 서비스 정보를 탐색하고, 소셜 로그인 버튼을 클릭하여 온보딩을 시작.

#### 1-2. 소셜 로그인 모달
**핵심 목표:** 사용자가 어떤 페이지에 있든 빠르고 쉽게 로그인할 수 있도록 지원.

**주요 구성 요소:** 팝업 형태의 모달창, Google, 카카오 로그인 버튼.

**사용자 인터랙션:** 버튼 클릭 시 Supabase Auth를 통해 각 소셜 플랫폼의 인증 프로세스로 연결.

#### 1-3. 프로필 설정 페이지 (온보딩 과정)
**핵심 목표:** 정확한 운세 분석에 필요한 최소한의 정보를 frictionless한 방식으로 수집.

**주요 구성 요소:**
- **진행 상태 바:** 전체 과정(예: 1/3, 2/3)을 시각적으로 표시
- **1단계 (필수):** 이름, 생년월일, 출생 시간 입력 폼. 각 필드마다 왜 필요한지에 대한 간단한 설명 제공 (예: "정확한 사주 분석을 위해 필요해요")
- **2단계 (선택):** MBTI, 성별 입력 폼. 나중에 설정할래요 또는 건너뛰기 버튼 제공

**사용자 인터랙션:** 폼 필드에 정보를 입력하고 다음 버튼을 눌러 진행. 입력 즉시 유효성 검사(Zod 활용) 피드백 제공.

### 2. 메인 네비게이션 페이지 (Main Navigation Pages)

#### 2-1. 🏠 홈 (대시보드)
**핵심 목표:** 개인화된 운세 정보를 요약하여 제공하고, 다른 핵심 기능으로의 접근성을 높이는 허브 역할.

**주요 구성 요소:**
- **상단 헤더:** [사용자 이름]님, 반가워요!
- **오늘의 운세 카드:** 오늘의 총운 점수, 핵심 키워드, 한 줄 요약을 담은 가장 큰 카드. 클릭 시 상세 페이지로 이동
- **위젯 그리드 (스크롤 가능):**
  - 나의 맞춤 운세: MBTI 주간 운세, 별자리 월간 운세 등을 담은 가로 스크롤 카드 리스트
  - 빠른 실행 도구: 타로카드, 관상 분석, 꿈해몽 아이콘 버튼
  - 지금 고민은?: 연애운, 취업운, 금전운 등 주제별 운세 바로가기
  - 스페셜 콘텐츠 추천: '연예인 궁합', '반려동물 사주' 등 흥미로운 콘텐츠를 주기적으로 노출

**사용자 인터랙션:** 스크롤, 각 카드 및 버튼 클릭을 통해 상세 페이지로 이동.

#### 2-2. 🧭 전체 운세 (탐색)
**핵심 목표:** 앱에서 제공하는 모든 운세 서비스를 체계적인 라이브러리 형태로 제공하여 사용자의 탐색을 도움.

**주요 구성 요소:**
- **상단 검색 바:** "찾고 싶은 운세가 있나요?" (예: '재물', '궁합')
- **카테고리 탭:** 데일리, 심층 분석, 인터랙티브, 전통 점술 등 정보 구조에 따른 탭 메뉴
- **운세 리스트:** 선택된 카테고리의 운세들이 아이콘, 이름, 짧은 설명을 포함한 카드 형태로 나열

**사용자 인터랙션:** 검색, 탭 전환, 스크롤, 카드 클릭.

#### 2-3. ✨ 스페셜 (주제별/흥미 운세)
**핵심 목표:** 특정 목적이나 재미를 위해 앱을 사용하는 사용자에게 맞춤형 콘텐츠를 집중적으로 노출.

**주요 구성 요소:**
- **배너/이벤트 섹션:** 시즌별 이벤트나 새로운 콘텐츠를 소개하는 배너
- **주제별 카테고리:** 💖 연애/궁합, 💼 직업/학업, 💰 재물/투자 등 명확한 아이콘과 함께 섹션 구분
- **흥미 위주 콘텐츠 리스트:** 연예인 궁합, 이름풀이 등을 매력적인 썸네일과 함께 제시

**사용자 인터랙션:** 카테고리 선택 및 콘텐츠 카드 클릭.

#### 2-4. 📊 나의 기록 (히스토리)
**핵심 목표:** 사용자가 자신의 운세 기록을 관리하고, 운의 흐름을 시각적으로 파악하도록 도움.

**주요 구성 요소:**
- **탭 메뉴:** 최근 기록, 통계 분석 2개의 탭으로 구성
- **최근 기록 탭:** 날짜, 운세 종류, 핵심 결과가 요약된 리스트. 날짜/종류별 필터 기능 제공
- **통계 분석 탭:**
  - 운세 점수 변화 그래프: 기간별(주간/월간) 오늘의 총운 점수 변화를 보여주는 라인 차트
  - 운세 카테고리 분석: 가장 많이 본 운세 종류를 보여주는 파이 차트

**사용자 인터랙션:** 필터링, 차트의 특정 데이터 포인트 터치 시 상세 정보 확인.

#### 2-5. 👤 프로필 (설정)
**핵심 목표:** 사용자 정보 수정 및 앱의 전반적인 설정을 관리하는 공간.

**주요 구성 요소:**
- **프로필 요약:** 프로필 사진, 이름, 이메일 표시. 프로필 수정 버튼
- **메뉴 리스트:**
  - 알림 설정 (푸시 알림 ON/OFF)
  - 프리미엄 구독 관리
  - 고객센터 / 문의하기
  - 이용약관 및 개인정보 처리방침
  - 로그아웃, 회원 탈퇴

**사용자 인터랙션:** 각 메뉴 항목 클릭하여 해당 기능으로 이동.

### 3. 핵심 운세 상세 페이지 (Core Fortune Detail Pages)

#### 3-1. 오늘의 총운 상세
**핵심 목표:** 오늘의 운세를 깊이 있고 실용적으로 제공하여 일상생활에 도움이 되도록 지원.

**주요 구성 요소:**
- **상단 요약:** 오늘의 운세 점수(100점 만점), 핵심 키워드, 한 줄 요약
- **세부 분석:** 총운, 연애운, 직업운, 금전운, 건강운을 각각 점수와 함께 상세 설명
- **오늘의 조언:** 실천 가능한 구체적인 행동 지침 제공
- **행운 아이템/색상:** 오늘 착용하면 좋은 색상이나 소지품 추천

#### 3-2. 사주팔자 상세 분석
**핵심 목표:** 복잡하고 방대한 사주 정보를 사용자가 이해하기 쉽도록 시각적이고 단계적으로 제공.

**주요 구성 요소:**
- **상단 요약:** "당신은 지혜로운 물(水)의 기운을 가진 사람입니다."와 같이 핵심 오행을 시각화하여 제시
- **탭 기반 콘텐츠:**
  - 총운: 타고난 성격, 장단점 등 종합적인 설명
  - 오행 분석: 오행(목화토금수)의 분포를 보여주는 레이더 차트나 바 차트
  - 인생 주기: 초년/중년/말년운을 시기별로 나누어 설명
  - 상세 풀이: 재물, 연애, 건강 등 주제별 상세 분석. (일부 내용은 프리미엄 기능으로 잠금 처리)
- **결과 공유/저장 버튼:** 페이지 하단에 고정

#### 3-3. 타로카드 (인터랙션 포함)
**핵심 목표:** 실제 타로를 보는 듯한 몰입감과 리추얼 경험을 제공.

**주요 구성 요소 (단계별 UI):**
- **1단계 (질문):** "마음속으로 질문에 집중하고 카드를 섞어주세요." 메시지와 함께 셔플 애니메이션
- **2단계 (선택):** 펼쳐진 카드 중 특정 개수(예: 3장)를 선택하도록 안내. 카드 위에 마우스를 올리면 빛나는 효과
- **3단계 (결과):** 선택된 카드의 이미지, 이름, 핵심 키워드(정/역방향 표시)를 상단에 보여줌. 하단에는 각 카드(과거, 현재, 미래 등)에 대한 상세 AI 해석을 제공

#### 3-4. 관상 분석 (인터랙션 포함)
**핵심 목표:** AI 분석 과정을 신기하고 재미있는 경험으로 제공.

**주요 구성 요소:**
- **사진 업로드 화면:** 정면 사진을 위한 가이드라인(안경X, 앞머리X)과 업로드 버튼
- **분석 중 화면:** 로딩 스피너와 함께 "얼굴의 기운을 분석하고 있습니다..."와 같은 메시지
- **결과 화면:**
  - 인터랙티브 이미지: 원본 사진 위에 눈, 코, 입 등 분석된 영역을 터치 가능한 포인트로 표시
  - 결과 텍스트: 포인트를 터치하면 해당 부위의 관상 해석이 동적으로 표시됨. 종합적인 성격, 재물운, 애정운 분석은 하단에 별도로 제공

#### 3-5. 꿈해몽 검색 및 결과
**핵심 목표:** 빠르고 정확하게 꿈의 의미를 찾을 수 있도록 도움.

**주요 구성 요소:**
- **검색 화면:** "꿈에서 본 키워드를 입력하세요" (예: '용', '이빨 빠지는 꿈') 문구가 있는 검색창. 인기 검색어 또는 최근 검색어 목록 표시
- **결과 화면:** 입력된 키워드와 관련된 다양한 해몽 리스트를 길몽, 흉몽 태그와 함께 제공. 각 항목 클릭 시 상세 해석 페이지로 이동

### 4. 기타 주요 페이지 (Other Key Pages)

#### 4-1. 프리미엄 구독 안내
**핵심 목표:** 사용자에게 프리미엄 기능의 가치를 명확히 전달하고 구독을 유도.

**주요 구성 요소:**
- **기능 비교표:** 무료 vs 프리미엄 기능을 한눈에 비교할 수 있는 표
- **프리미엄 전용 기능:** 상세 사주 분석, 광고 제거, 무제한 타로카드 등
- **구독 플랜:** 월간/연간 구독 옵션과 가격 정보
- **구독 혜택:** 첫 달 무료 체험, 특별 할인 등

#### 4-2. 결과 공유 모달
**핵심 목표:** 운세 결과를 소셜 미디어에 쉽고 예쁘게 공유할 수 있도록 지원.

**주요 구성 요소:**
- **공유 이미지 미리보기:** 운세 결과가 시각적으로 정리된 공유용 이미지
- **공유 플랫폼 선택:** Instagram, Facebook, KakaoTalk, Twitter 등 주요 SNS 버튼
- **텍스트 편집:** 공유할 때 함께 올릴 메시지 편집 기능
- **이미지 저장:** 공유용 이미지를 기기에 저장하는 기능
