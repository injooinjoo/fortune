## 💬 **사용법: 자연스럽게 말하기만 하면 끝!**

### ✨ **폰트/텍스트 관련**
```
"메인 화면 글자가 너무 작아요"
"제목 폰트 크기를 좀 더 크게 해주세요"
"텍스트가 읽기 어려워요"
"타이포그래피 개선이 필요해요"
```
→ **자동 분류**: font 카테고리 🔤

### 🎨 **색상/테마 관련**
```
"다크모드 색상이 잘 안 보여요"
"배경색이 너무 밝아요"
"버튼 색깔을 바꿔주세요"
"테마 색상을 조정하고 싶어요"
```
→ **자동 분류**: color 카테고리 🎨

### ✨ **애니메이션 관련**
```
"페이지 전환이 너무 딱딱해요"
"부드러운 애니메이션을 추가해주세요"
"움직임이 어색해요"
"트랜지션 효과가 필요해요"
```
→ **자동 분류**: animation 카테고리 ✨

### 📐 **레이아웃/배치 관련**
```
"버튼 간격이 너무 좁아요"
"여백을 늘려주세요"
"배치가 이상해요"
"레이아웃을 조정해주세요"
```
→ **자동 분류**: layout 카테고리 📐

### ♿ **접근성/사용성 관련**
```
"버튼이 너무 작아서 터치하기 어려워요"
"누르기 힘들어요"
"클릭 영역을 넓혀주세요"
"사용하기 불편해요"
```
→ **자동 분류**: accessibility 카테고리 ♿

### 🧭 **내비게이션 관련**
```
"뒤로가기 버튼이 없어요"

```
→ **자동 분류**: navigation 카테고리 🧭

## 🤖 **Claude Code가 자동으로 하는 일**

### 1️⃣ **자연어 분석**
- 키워드 감지로 카테고리 자동 분류
- 우선순위 자동 설정 (급해/중요 → 높음, 접근성 → 중간, 기본 → 낮음)
- 제목 자동 추출 (첫 문장 또는 요약)

### 2️⃣ **JIRA 자동 등록**
- 적절한 라벨 자동 태깅
- 상세 설명 템플릿 자동 생성
- 작업 단계 가이드 포함

### 3️⃣ **실시간 피드백**
```
🤖 자연어 UX 요청 감지됨!
📝 제목: 버튼이 너무 작아서 터치하기 어려워요
🏷️ 카테고리: accessibility
⚡ 우선순위: medium

✅ JIRA 이슈 KAN-40 생성 완료!
🔗 https://beyond-app.atlassian.net/browse/KAN-40
```

## 🎯 **고급 기능**

### 🔄 **작업 완료 연동**
작업 완료 시에도 자연어로:

```
"KAN-40 버튼 크기 작업 끝났어요"
→ 자동으로 Git 커밋 + JIRA Done 처리

"폰트 크기 조정 완료했습니다"
→ 관련 이슈 자동 검색 후 완료 처리
```

### 📊 **우선순위 자동 감지**
- **"급해", "빨리", "중요"** → 높은 우선순위
- **"접근성", "장애", "불편"** → 중간 우선순위
- **기본** → 낮은 우선순위

### 🏷️ **자동 라벨링**
- 카테고리별 전문 라벨
- `auto-generated` 라벨로 자동 생성 표시
- 우선순위 라벨 자동 추가
