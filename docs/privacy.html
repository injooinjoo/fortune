Privacy Policy

ZPZG 개인정보처리방침

시행일: 2026년 3월 18일. 이 문서는 ZPZG 앱과 관련 웹 페이지에서 어떤 정보를 수집하고 어떻게 사용하는지 설명합니다.

1. 수집하는 정보

  • 계정 정보: 이메일 주소, 닉네임, 사용자 식별자
  • 서비스 입력 정보: 생년월일, 출생 시간/지역, 사용자가 직접 입력한 운세 관련 질문
  • 선택 정보: 프로필 사진, 얼굴 분석용 사진, 음성 입력 내용, 위치, 캘린더 일정
  • 결제 정보: 토큰 구매 및 구독 상태 확인에 필요한 구매 이력
  • 서비스 이용 정보: 화면 조회, 기능 사용 기록, 기본적인 기기/앱 진단 정보

2. 이용 목적

  • 회원 가입, 로그인, 계정 보호, 고객 문의 대응
  • 운세·관계·성향·생활 인사이트 기능 제공
  • 토큰/구독 구매 확인 및 결제 처리
  • 서비스 품질 분석과 안정성 개선
  • 법령 준수와 부정 이용 방지

3. 권한 사용 안내

  • 카메라: 사용자가 직접 얼굴 사진을 촬영할 때만 사용합니다.
  • 사진 보관함: 사용자가 직접 선택한 사진만 업로드합니다.
  • 마이크/음성 인식: 음성 입력 기능을 사용할 때만 요청합니다.
  • 위치: 날씨/지역 기반 인사이트를 사용할 때만 사용합니다.
  • 캘린더: 사용자가 일정 저장 기능을 실행할 때만 사용합니다.

선택 권한은 해당 기능을 직접 실행할 때만 요청되며, 거부해도 다른 핵심 기능은 계속 이용할 수 있습니다.

4. 제3자 제공 및 처리 위탁

  • Supabase: 계정 관리, 데이터 저장, 서비스 운영
  • Google Firebase Analytics: 비식별 사용 통계 분석
  • Apple App Store / Google Play: 인앱 결제 처리 및 구매 확인

광고 추적을 위한 ATT/IDFA 기반 추적은 사용하지 않으며, 개인정보를 판매하지 않습니다.

5. 보관 기간과 삭제

계정 정보는 회원 탈퇴 시까지 보관합니다. 결제 관련 기록은 관련 법령과 스토어 정책에 따라 필요한 기간 동안 보관할 수 있습니다. 사용자는 앱 내 설정에서 계정 삭제를 요청할 수 있습니다.
