서비스는 원칙적으로 개인정보 수집 및 이용목적이 달성된 후에는 해당 정보를 지체 없이 파기합니다. 회원 정보: 회원 탈퇴 시까지 결제 정보: 전자상거래법에 따라 5년간 보관 서비스 이용 기록: 통신비밀보호법에 따라 3개월간 보관

Fortune 앱은 서비스 개선을 위해 다음의 분석 서비스를 이용합니다: • Firebase Analytics: 앱 사용 분석 및 오류 추적 수집되는 정보: • 기기 정보 (기기 모델, OS 버전) • 앱 사용 패턴 (화면 조회, 기능 사용 빈도) • 오류 로그 (앱 충돌 시 디버깅용) 이 정보는 익명화되어 개인을 식별할 수 없으며, 서비스 품질 향상에만 활용됩니다.

Fortune 앱은 다음의 기기 권한을 사용합니다: • 카메라: 관상 분석 기능에서 얼굴 사진 촬영 • 사진 보관함: 관상 분석용 사진 선택 및 운세 이미지 저장 • 마이크: 꿈 해몽 기능에서 음성으로 꿈 내용 입력 • 음성 인식: 음성을 텍스트로 변환하여 꿈 해몽 제공 • 위치: 현재 위치 기반 날씨 정보를 운세에 반영 • 캘린더: 중요한 운세 날짜 저장 및 일정 분석 기반 맞춤 운세 각 권한은 해당 기능 사용 시에만 요청되며, 권한을 거부해도 해당 기능 외의 앱 사용에는 제한이 없습니다.
