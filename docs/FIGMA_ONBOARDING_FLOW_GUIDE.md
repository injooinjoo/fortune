# Fortune App - Onboarding Flow Design Guide

## 전체 구조
- **총 4단계**: 이름 → 생년월일 → 성별 → 지역
- **Frame 크기**: iPhone 14 Pro (393 × 852)
- **배경색**: #FFFFFF

## 공통 컴포넌트

### 1. Progress Bar
```
위치: 상단
높이: 4px
배경색: #E5E7EB
진행 색상: #000000
```
- **너비**: 화면 전체
- **진행률**: currentStep / totalSteps * 100%
- **애니메이션**: 300ms ease-in-out

### 2. 뒤로가기 버튼
```
위치: 좌측 상단 (16px, 16px)
크기: 48×48px (터치 영역)
아이콘 크기: 24×24px
```
- **아이콘**: arrow_back_ios
- **색상**: #111827
- **배경**: 투명

### 3. 다음/완료 버튼 (공통)
- **위치**: 하단 고정
- **너비**: 100% - 48px (좌우 24px 패딩)
- **높이**: 56px
- **하단 간격**: Safe Area + 16px
- **배경색**: #000000 (활성화) / #E5E7EB (비활성화)
- **텍스트 색상**: #FFFFFF (활성화) / #9CA3AF (비활성화)
- **모서리 반경**: 28px
- **폰트**: NotoSansKR SemiBold, 18px

## 상태 관리

### 버튼 활성화 조건
- **Step 1**: 이름 2자 이상 입력
- **Step 2**: 날짜 선택 완료
- **Step 3**: 성별 선택 완료
- **Step 4**: 지역 선택 완료

### 전환 애니메이션
- **페이지 전환**: slideX + fadeIn (300ms)
- **Progress Bar**: width 애니메이션 (300ms)

## Figma 구현 팁

### 1. 컴포넌트 구조
```
Onboarding Flow
├── Progress Bar (Component)
├── Back Button (Component)
├── Primary Button (Component with states)
├── Input Field (Component with states)
└── Selection Card (Component with states)
```

### 2. Auto Layout
- 각 스텝을 Frame으로 구성
- Vertical Auto Layout 사용
- Space between으로 버튼 하단 고정

### 3. Interactive Components
- Input Field: default → focused → filled
- Selection Card: default → selected
- Button: disabled → enabled → pressed

### 4. 프로토타입
- 각 단계별 Flow 연결
- Smart Animate 사용
- Progress Bar 애니메이션
