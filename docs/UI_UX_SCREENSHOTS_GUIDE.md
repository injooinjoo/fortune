## 개요

Fortune 앱은 총 100개 이상의 화면으로 구성되어 있으며, 크게 다음과 같은 섹션으로 분류됩니다:

- **인증 및 온보딩**: 6개 화면
- **메인 네비게이션**: 5개 탭
- **운세 서비스**: 60개+ 화면
- **인터랙티브 기능**: 8개 화면
- **결제 및 구독**: 7개 화면
- **사용자 관리**: 8개 화면
- **관리자**: 3개 화면

## 주요 사용자 플로우

### 1. 신규 사용자 온보딩 플로우
```
[랜딩 페이지] → [로그인 선택] → [소셜 로그인] → [프로필 설정] → [생년월일 입력] → [MBTI 선택] → [홈 화면]
```

**스크린샷 필요 화면:**
- `/` - 랜딩 페이지 (로고, 소개 텍스트, 시작하기 버튼)
- `/auth/selection` - 로그인 방법 선택 (Google, Kakao, Email)
- `/onboarding` - 환영 메시지 및 프로필 시작
- `/onboarding/profile` - 이름, 성별 입력
- `/onboarding/birthdate` - 생년월일 입력 (달력 UI)
- `/onboarding/mbti` - MBTI 16가지 타입 선택 그리드
- `/home` - 메인 홈 화면

### 2. 운세 보기 플로우
```
[운세 카테고리] → [세부 선택] → [정보 입력] → [토큰 확인] → [결과 보기] → [공유/저장]
```

**스크린샷 필요 화면:**
- `/fortune` - 운세 메인 허브 (카테고리 그리드)
- `/fortune/daily` - 일일 운세 (정보 입력 폼)
- 토큰 차감 확인 모달
- 운세 결과 화면 (애니메이션 포함)
- 공유 모달 (카카오톡, 인스타그램 등)

### 3. 토큰 구매 플로우
```
[토큰 부족 알림] → [패키지 선택] → [결제 수단] → [결제 진행] → [구매 완료]
```

**스크린샷 필요 화면:**
- 토큰 부족 경고 모달
- `/payment/tokens` - 토큰 패키지 선택
- 결제 수단 선택 (Stripe, TossPay)
- 결제 진행 화면
- `/payment/success` - 구매 완료 화면

## 화면별 상세 설명

### 🏠 메인 네비게이션 (하단 탭)

#### 1. 홈 탭 (`/home`)
- **헤더**: 로고, 토큰 잔액, 알림 아이콘
- **메인 콘텐츠**:
  - 오늘의 운세 카드
  - 추천 운세 캐러셀
  - 인기 운세 리스트
  - 이벤트 배너
- **하단**: 5개 탭 네비게이션 바

#### 2. 운세 탭 (`/fortune`)
- **헤더**: "운세" 타이틀, 검색 아이콘
- **메인 콘텐츠**:
  - 카테고리별 그리드 (10개 카테고리)
  - 각 카테고리 아이콘과 라벨
  - 신규/인기 뱃지
- **특징**: 그리드 레이아웃, 카테고리별 색상 구분

#### 3. 관상 탭 (`/physiognomy`)
- **헤더**: "AI 관상" 타이틀
- **메인 콘텐츠**:
  - 카메라 뷰 또는 사진 업로드
  - 얼굴 인식 가이드라인
  - 분석 시작 버튼
- **결과 화면**: 얼굴 특징별 분석 결과

#### 4. 프리미엄 탭 (`/premium`)
- **헤더**: "Premium" 타이틀, 왕관 아이콘
- **메인 콘텐츠**:
  - 현재 구독 상태
  - 프리미엄 혜택 리스트
  - 구독 플랜 카드 (Basic, Premium, Enterprise)
  - CTA 버튼
- **특징**: 그라데이션 배경, 프리미엄 느낌의 UI

#### 5. 프로필 탭 (`/profile`)
- **헤더**: 프로필 사진, 이름
- **메인 콘텐츠**:
  - 토큰 잔액 카드
  - 메뉴 리스트:
    - 프로필 편집
    - 운세 히스토리
    - 즐겨찾기
    - 알림 설정
    - 고객 지원
    - 로그아웃
- **특징**: 리스트 아이템 with 아이콘

### 🔮 주요 운세 화면

#### 사주팔자 (`/fortune/saju`)
- **입력 필드**:
  - 이름
  - 생년월일 (양력/음력 선택)
  - 태어난 시간
  - 성별
- **특징**:
  - 한국식 날짜 선택기
  - 음력 변환 기능
  - 툴팁 설명

#### 타로카드 (`/interactive/tarot`)
- **메인 화면**:
  - 카드 뒷면 22장 배열
  - 카드 선택 애니메이션
  - 선택된 카드 하이라이트
- **결과 화면**:
  - 선택한 카드 이미지
  - 카드 의미 설명
  - 전체 해석

#### 오늘의 운세 (`/fortune/today`)
- **콘텐츠**:
  - 종합 운세 점수 (시각적 차트)
  - 카테고리별 운세:
    - 애정운
    - 금전운
    - 건강운
    - 직장운
  - 행운의 숫자/색상/방향
  - 주의사항

### 💳 결제 관련 화면

#### 토큰 구매 (`/payment/tokens`)
- **패키지 선택**:
  - Small (10개) - ₩1,100
  - Medium (60개) - ₩5,500
  - Large (150개) - ₩11,000
  - 각 패키지별 보너스 표시
- **결제 수단**:
  - 신용카드 (Stripe)
  - 토스페이
  - 네이버페이 (예정)

#### 구독 관리 (`/subscription`)
- **현재 플랜 정보**:
  - 플랜 이름 및 가격
  - 다음 결제일
  - 사용 중인 혜택
- **플랜 변경**:
  - 업그레이드/다운그레이드 옵션
  - 구독 취소

### 📊 대시보드 화면

#### 사용자 대시보드 (`/dashboard`)
- **통계 카드**:
  - 이번 달 본 운세 수
  - 토큰 사용량
  - 즐겨찾기한 운세
- **차트**:
  - 일별 운세 조회 그래프
  - 카테고리별 사용 비율

#### 관리자 대시보드 (`/admin`)
- **실시간 통계**:
  - 활성 사용자 수
  - 일일 수익
  - API 호출 수
  - 에러율
- **관리 기능**:
  - 사용자 관리
  - 토큰 사용량 모니터링
  - Redis 상태

## 컴포넌트 인터랙션

### 1. 토큰 잔액 표시 컴포넌트
- **위치**: 모든 주요 화면 상단
- **인터랙션**:
  - 탭하면 토큰 히스토리 모달
  - 잔액 부족시 빨간색 표시
  - 충전 버튼 바로가기

### 2. 운세 결과 공유 모달
- **트리거**: 결과 화면의 공유 버튼
- **옵션**:
  - 이미지로 저장
  - 카카오톡 공유
  - 인스타그램 스토리
  - 링크 복사
- **특징**: 워터마크 자동 추가

### 3. 프로필 완성도 알림
- **조건**: 프로필 미완성시 주기적 표시
- **내용**:
  - 완성도 퍼센트 표시
  - 미입력 항목 안내
  - "지금 완성하기" CTA

### 4. 로딩 상태
- **운세 생성중**:
  - 맥동하는 원형 로더
  - 재치있는 로딩 메시지
  - 예상 시간 표시
- **결제 처리중**:
  - 전체 화면 로더
  - 진행 단계 표시

### 5. 에러 처리
- **네트워크 에러**:
  - 토스트 메시지
  - 재시도 버튼
- **토큰 부족**:
  - 모달 팝업
  - 충전 유도 CTA
