- 긍정적이고 건설적으로
- 과학적이고 투명하게

## 🎯 운세별 개별 전략

### 1. 데일리 운세 그룹

#### 📅 daily (오늘의 운세)
**현재 상태**: ✅ GPT 연동 완료, 캐싱 구현
**수집 정보**: 이름, 생년월일, 성별(선택), MBTI(선택)
**표시 정보**: 총운/애정운/재물운/건강운 점수, 행운시간, 행운색상
**개선 방안**:
- 프로필 정보 자동 로드 강화
- 시간대별 운세 변화 추가
- 푸시 알림 연동 (오전 7시)

#### ⏰ hourly (시간별 운세)
**현재 상태**: ✅ GPT 연동 완료
**수집 정보**: 이름, 생년월일, 생시(필수)
**표시 정보**: 24시간 운세 그래프, 최적/주의 시간대
**특징**: 생시 정보를 활용한 정밀 분석
**개선 방안**:
- 실시간 업데이트 기능
- 알람 설정 기능
- 캘린더 연동

## 🔧 공통 패턴 및 개선사항

### 재사용 가능한 컴포넌트
1. **UserInfoForm**: 이름, 생년월일 입력 표준화
2. **FortuneResultDisplay**: 운세 결과 표시 통일
3. **FortuneScoreCard**: 점수 표시 컴포넌트
4. **LuckyItemsGrid**: 행운 아이템 그리드
5. **ShareButtons**: SNS 공유 버튼 세트

### 공통 데이터 구조
```typescript
interface BaseFortuneInput {
  name: string;
  birth_date: string;
  birth_time?: string;
  gender?: 'male' | 'female';
}

interface BaseFortuneResult {
  overall_score: number;
  summary: string;
  details: string;
  lucky_items?: {
    color?: string;
    number?: number;
    direction?: string;
    time?: string;
  };
  advice?: string;
}
```

### API 통합 전략
1. **배치 요청**: 관련 운세들을 묶어서 요청
   - 데일리 패키지: daily + hourly + today + tomorrow
   - 전통 패키지: saju + tojeong + palmistry
   - 관계 패키지: love + marriage + compatibility

2. **캐싱 전략**:
   - 변하지 않는 운세: 365일 캐싱 (saju, palmistry)
   - 일일 운세: 24시간 캐싱
   - 시간별 운세: 1시간 캐싱

3. **프로필 연동**:
   - 로그인 사용자: 자동 정보 로드
   - 비로그인: localStorage 활용
   - 정보 재사용률 향상
