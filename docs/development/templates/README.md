# Hard Block Report Templates

이 디렉토리는 Codex/Claude 공통 Hard Block 흐름에서 사용할 보고서 템플릿 모음입니다.

## Files
- `RCA_REPORT_TEMPLATE.md`
- `DISCOVERY_REPORT_TEMPLATE.md`
- `VERIFY_REPORT_TEMPLATE.md`

## Usage
1. 버그/에러 수정 전 `RCA_REPORT_TEMPLATE.md` 작성
2. 새 코드 작성 전 `DISCOVERY_REPORT_TEMPLATE.md` 작성
3. 수정 완료 후 `VERIFY_REPORT_TEMPLATE.md` 작성
