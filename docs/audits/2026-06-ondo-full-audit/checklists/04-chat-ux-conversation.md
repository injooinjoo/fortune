# 공통 운영 원칙

- 온도 앱 전체 검토의 일부로 이 체크리스트를 사용한다.
- 1차 작업은 **QA/리뷰/증거 수집/수정 제안**이 목적이다.
- 코드 수정은 하지 않는다. 수정은 취합 후 별도 Fix Agent 작업으로 진행한다.
- 모든 발견 항목은 P0/P1/P2/P3로 분류한다.
- 단순 의견이 아니라 증거 기반으로 보고한다.
- 문서와 코드가 다르면 코드/실제 동작을 우선 증거로 삼는다.
- 시뮬레이터 성공을 실기기 성공으로 간주하지 않는다.
- 서버 작업 완료를 유저 화면 성공으로 간주하지 않는다.

## 심각도 기준

- P0: 앱 사용 불가, 결제/토큰 손실, 보안/개인정보 문제, App Store 즉시 리젝 가능
- P1: 핵심 기능 실패, 사용자 신뢰 크게 저하, 주요 전환/매출 손상
- P2: UX 불편, 특정 경로에서 실패, 반복 버그 가능성
- P3: 개선 제안, polish, 미세 최적화

# 4. Chat UX & Conversation Reviewer

## 검토 목표
- 채팅 경험이 감성적이고 자연스럽고 신뢰 가능한지 확인한다.
- 한국어 대화 품질, 캐릭터 톤, 메시지 UI, 멀티모달 메시지 경험을 검토한다.

## 체크리스트

### 1. 첫 진입 경험
- 채팅방 진입이 자연스러운가?
- 첫 메시지/인사말이 어색하지 않은가?
- 유저가 무엇을 해야 하는지 명확한가?

### 2. 메시지 작성 UX
- 입력창 위치가 적절한가?
- 키보드와 composer 간격이 자연스러운가?
- 전송 버튼 상태가 명확한가?
- 빈 메시지/긴 메시지/줄바꿈 처리가 자연스러운가?

### 3. 답변 경험
- 답변 로딩 상태가 명확한가?
- 타이핑 표시가 있는가?
- 실패/재시도 UX가 있는가?
- 중복 답변/늦은 답변 처리 UX가 자연스러운가?

### 4. 캐릭터 말투
- 캐릭터별 톤이 구분되는가?
- 한국어가 자연스러운가?
- 썸남/친구/상담자/하늘이 톤이 일관적인가?
- 방어적이거나 싸움 거는 답변이 없는가?
- 너무 AI스럽거나 generic하지 않은가?

### 5. 메시지 렌더링 / 멀티모달
- 말풍선 정렬이 자연스러운가?
- 시간 표시가 적절한가?
- 긴 텍스트/이모지/공백/링크 처리 문제가 없는가?
- 이미지 메시지 정렬/캡션/로딩/실패 처리가 적절한가?
- 오디오 메시지 녹음/재생/취소/방 전환 처리가 안전한가?
- 앱 재시작 후 오디오/이미지 메시지가 복원되는가?

### 6. 채팅 리스트/헤더
- 최신 메시지 preview가 정확한가?
- 캐릭터 아바타/이름/프로필 진입이 자연스러운가?
- 읽음/안읽음 상태가 직관적인가?
