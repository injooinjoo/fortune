## 1. 프로젝트 개요

### 앱 소개
- **앱 이름**: Fortune (운세 앱)
- **플랫폼**: iOS, Android (Flutter)
- **주요 기능**: 타로, 사주, 관상, 궁합, MBTI 운세 등 30+ 운세 서비스
- **타겟 사용자**: 20-40대, 운세/심리테스트에 관심 있는 사용자

- Toss 디자인 시스템 기반 (깔끔한 미니멀 스타일)
- 한국 전통 요소 일부 포함 (한지 배경, 오방색)

### 리디자인 방향 (논의 필요)
- [ ] 현재 스타일 유지 + 개선
- [ ] 완전히 새로운 컨셉

## 2. 파일 형식 가이드

### 벡터 에셋 (아이콘, 로고)
| 형식 | 용도 | 비고 |
|------|------|------|
| **SVG** | 기본 제공 형식 | 필수 |
| PDF | iOS 전용 | 선택 |

### 이미지 에셋 (일러스트, 배경)
| 형식 | 해상도 | 용도 |
|------|--------|------|
| **PNG** | @1x, @2x, @3x | 기본 |
| WebP | @2x, @3x | 용량 최적화 |

### 해상도 가이드
```
@1x: 기준 크기
@2x: 기준 x 2 (대부분의 기기)
@3x: 기준 x 3 (고해상도 기기)

예시: 24x24 아이콘
- icon@1x.png: 24x24px
- icon@2x.png: 48x48px
- icon@3x.png: 72x72px
```

### 애니메이션
| 형식 | 용도 |
|------|------|
| **Lottie (JSON)** | 복잡한 애니메이션 |
| GIF | 간단한 애니메이션 |
| APNG | 투명 배경 필요시 |

### 5.3 네비게이션 아이콘 (5개)

하단 탭바에 사용, **2가지 상태** 필요

| # | 아이콘명 | 현재 | 디자인 방향 | 상세 설명 |
|---|---------|------|------------|----------|
| 1 | **home** | Material `home` | 집 모양 | 지붕 + 문, 단순한 집 실루엣. 선형(outlined) / 면형(filled) |
| 2 | **fortune** | Material `auto_awesome` | ⭐ 별/수정구 | 반짝이는 별 또는 수정구 모양. 운세/마법 느낌. 3개 별이 모인 형태 |
| 3 | **trend** | Material `trending_up` | 📈 상승 그래프 | 우상향 화살표/그래프 선. 트렌드/인기 의미 |
| 4 | **premium** | Material `workspace_premium` | 👑 왕관/메달 | 프리미엄 느낌의 왕관 또는 리본 달린 메달 |
| 5 | **profile** | Material `person` | 👤 사람 | 머리+어깨 실루엣. 원 안에 상반신 |

#### 네비게이션 아이콘 상세 스펙

**공통 스펙**:
- 캔버스: 24x24px
- 선 두께: 1.5px (outlined), 채움 (filled)
- 모서리: 2px radius
- 패딩: 2px 내부 여백

**상태별 디자인**:
```
┌─────────────────────────────────────────────────────┐
│  Default (비활성)     │  Selected (활성)            │
├─────────────────────────────────────────────────────┤
│  • Outlined (선형)    │  • Filled (면형)            │
│  • 색상: Gray500      │  • 색상: Primary            │
│  • 선 두께: 1.5px     │  • 완전 채움               │
└─────────────────────────────────────────────────────┘
```

**각 아이콘 비주얼 가이드**:

```
1. HOME (홈)
   ┌───────┐
   │  /\   │  지붕 (삼각형)
   │ /  \  │
   │ ├──┤  │  집 본체 (사각형)
   │ │  │  │  문 (가운데 작은 사각형)
   └───────┘

2. FORTUNE (운세)
   ┌───────┐
   │  ✦    │  큰 별 (중앙)
   │ ✧  ✧  │  작은 별 2개 (좌우 또는 주변)
   │       │  반짝임 효과 (선 4개)
   └───────┘

   ┌───────┐
```

### 5.4 액션 아이콘 (15개)

일반 UI 액션에 사용, **1가지 상태** (색상으로 상태 표현)

| # | 아이콘명 | 비주얼 | 상세 설명 |
|---|---------|--------|----------|

| 4 | **bookmark** | `🔖` | 저장. 리본/깃발 형태, 하단 V자 컷 |
| 5 | **settings** | `⚙` | 설정. 톱니바퀴 6-8개 이빨, 중앙 원 |
| 6 | **edit** | `✏` | 수정. 연필 45도 기울임, 지우개 부분 표시 |
| 7 | **delete** | `🗑` | 삭제. 휴지통 본체 + 뚜껑, 세로선 2-3개 |
| 8 | **search** | `🔍` | 검색. 원 + 우하단 대각선 손잡이 |
| 9 | **filter** | `⏳` | 필터. 깔때기/역삼각형 형태, 위 넓고 아래 좁음 |

| 11 | **add** | `+` | 추가. 플러스 기호, 선 두께 2px |
| 12 | **remove** | `−` | 제거. 마이너스 기호, 가로선만 |
| 13 | **check** | `✓` | 완료. 체크마크, 좌하단→중앙→우상단 |

| # | 타입 | 용도 | 스타일 |

| 5 | **Floating** | FAB, 떠있는 버튼 | 그림자 + 원형/둥근사각 |

| 크기 | 높이 | 패딩 (좌우) | 폰트 |

| Large | 56px | 32px | buttonLarge (19pt) |
| Medium | 48px | 24px | buttonMedium (18pt) |
| Small | 40px | 16px | buttonSmall (17pt) |
| XSmall (Chip) | 32px | 12px | labelSmall (13pt) |

| 상태 | 설명 | 시각적 변화 |
|------|------|------------|
| Default | 기본 상태 | 기본 스타일 |
| Hover | 마우스 오버 | 밝기 +5% |
