## 🎴 사주팔자 (四柱八字)

### 📖 사주팔자란?

사주팔자는 태어난 년, 월, 일, 시의 네 기둥(四柱)과 
각각의 천간지지 여덟 글자(八字)로 운명을 해석하는 동양 최고의 운명학입니다.

### 🏛️ 사주의 구성

```
┌────────────────────────────────────────────────┐
│                   사주명식                      │
├─────────┬─────────┬─────────┬─────────────────┤
│  年柱   │  月柱   │  日柱   │  時柱           │
├─────────┼─────────┼─────────┼─────────────────┤
│ 甲 子   │ 丙 寅   │ 戊 辰   │ 庚 午           │
│ (년간)  │ (월간)  │ (일간)  │ (시간)          │
│ (년지)  │ (월지)  │ (일지)  │ (시지)          │
└─────────┴─────────┴─────────┴─────────────────┘

천간(天干): 甲乙丙丁戊己庚辛壬癸 (10개)
지지(地支): 子丑寅卯辰巳午未申酉戌亥 (12개)
```

### 💡 사주 개운법

#### 1. 오행 보완 전략
```
부족한 火(화) 보완법:
• 의상: 빨간색, 분홍색 계열
• 음식: 매운 음식, 붉은 채소
• 방향: 남쪽 향해 활동
• 시간: 오전 11시-오후 1시 활동
• 계절: 여름철 적극 활동
```

#### 2. 용신(用神) 활용법
- **용신**: 사주에서 가장 도움이 되는 오행
- **기신**: 사주에서 해로운 오행

```
당신의 용신: 火(화)
활용법: 
- 이름에 火 의미 글자 사용
- 南향 집 선택
- 붉은 계열 인테리어
- 열정적인 활동 추구
```

## 📜 토정비결 (土亭秘訣)

매년 정월에 보는 한 해 운세의 정석입니다.

```
│ 🌻  │ 🌊  │ ☀️  │ 🌾  │

├─────┼─────┼─────┼─────┤
│ 9월 │10월 │11월 │12월 │
│ 🍂  │ 🍁  │ 🌰  │ ❄️  │
```

### 📊 괘상 해석 예시

```
상괘: ☰ 건(乾) - 하늘

━━━━━━━━━━━━━━━━━━

해석: 
"하늘과 땅이 통하지 않는 형상으로
```

## 🖐️ 손금 (手相)

### 운명이 새겨진 손바닥

손금은 타고난 운명과 성격, 미래를 
손바닥의 선과 구릉으로 해석합니다.

### 🤚 주요 손금선 분석

```
        중지
         │
    약지─┼─검지
         │
  감정선 ═══════════
         │
  두뇌선 ═══════════
         │ 
  생명선 ╱
        ╱
      ╱
```

### 📊 손금선별 의미

| 선 | 의미 | 좋은 형태 | 해석 |
|---|-----|----------|------|
| 생명선 | 건강, 체력 | 길고 선명 | 장수, 활력 |
| 두뇌선 | 지능, 재능 | 곧고 깊음 | 명석함 |
| 감정선 | 애정, 감성 | 완만한 곡선 | 풍부한 감정 |
| 운명선 | 직업, 성공 | 똑바른 선 | 안정된 인생 |
| 태양선 | 명예, 인기 | 선명함 | 성공과 명성 |
| 결혼선 | 결혼, 애정 | 1-2개 선명 | 행복한 결혼 |

### 🏔️ 구릉(丘) 분석

```
┌─────────────────────┐
│ 목성구  토성구  태양구 │
│   💪     🎯     ⭐   │
├─────────────────────┤
│        화성구        │
│         🔥          │
├─────────────────────┤
│ 금성구        월구   │
│   💕          🌙    │
└─────────────────────┘
```

각 구릉의 발달 정도:
- 볼록하면: 해당 기운 강함
- 평평하면: 보통
- 움푹하면: 부족함

성격과 운명을 읽는 전통 상법입니다.

### 🎭 12궁 관상도

```
┌─────────────────────┐
│  ① 명궁 (이마중앙)   │ 전체운
├──────┬──────┬───────┤
│②부모│③관록│④부모  │ 가족/직업
├──────┼──────┼───────┤
│⑤형제│ 👁️👁️ │⑥형제  │ 형제운
├──────┼──────┼───────┤
│⑦처첩│  👃  │⑧처첩  │ 배우자운
├──────┴──────┴───────┤
│    ⑨ 자녀궁 (인중)   │ 자녀운
├─────────────────────┤
│    ⑩ 재백궁 (턱)    │ 재물운
└─────────────────────┘
```

### 📊 얼굴형별 성격과 운세

## 🛡️ 살풀이

### 악운을 물리치는 전통 비법

살(煞)은 운명에 부정적 영향을 주는 
에너지로, 이를 풀어내는 것이 살풀이입니다.

### 👻 주요 살(煞)의 종류

```
┌─────────────────────────────┐
│ 삼재살 | 3년 주기 액운     │
│ 역마살 | 이동수, 변동      │
│ 도화살 | 이성 문제         │
│ 백호살 | 사고수, 다툼      │
│ 천을귀인 | 귀인 도움 (吉)  │
└─────────────────────────────┘
```

### 🔮 살풀이 방법

| 살 종류 | 증상 | 해결법 | 부적/행운템 |
|--------|-----|-------|------------|
| 삼재살 | 3년간 운 저하 | 절 방문, 기도 | 삼재부적 |
| 역마살 | 불안정, 이동 | 안정 추구 | 청동 거북이 |
| 도화살 | 이성 문제 | 절제, 수양 | 정절부적 |
| 백호살 | 다툼, 사고 | 조심, 양보 | 호랑이 부적 |
