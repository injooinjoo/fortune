# 영혼(Soul) 시스템 정책 문서

## 개요
Fortune 앱의 토큰 시스템을 "영혼(Soul)" 시스템으로 리브랜딩합니다. 이는 사용자의 운세 활동을 경험치처럼 누적하는 개념으로 전환하는 것입니다.

## 핵심 변경사항

### 1. 개념 변경
- **기존**: 토큰을 소비하여 운세를 확인
- **변경**: 운세를 확인하면 영혼을 획득 (경험치 개념)
- **명칭**: 토큰(Token) → 영혼(Soul)

### 2. 시스템 작동 방식

#### 영혼 획득 방법
- 일반 운세 확인 시 영혼 획득
- 일일 무료 운세로 영혼 획득
- 특별 이벤트 참여로 영혼 획득
- 광고 시청 후 영혼 획득

#### 영혼 소비 방법
- AI 기반 프리미엄 운세 이용 시
- 고급 영상 운세 콘텐츠 이용 시
- 특별 분석 리포트 생성 시

### 3. 운세 분류

#### A. 영혼을 주는 운세 (무료)
기본적인 운세들로, 사용자가 이용하면 영혼을 획득합니다:

**기본 운세 (1-2 영혼 획득)**
- 오늘의 운세 (daily)
- 내일의 운세 (tomorrow)
- 주간 운세 (weekly)
- 행운의 색깔 (lucky-color)
- 행운의 숫자 (lucky-number)
- 행운의 음식 (lucky-food)
- 행운의 아이템 (lucky-items)
- 탄생석 운세 (birthstone)
- 혈액형 운세 (blood-type)
- 띠 운세 (zodiac-animal)
- 별자리 운세 (zodiac)

**중급 운세 (3-5 영혼 획득)**
- 연애운 (love)
- 직업운 (career)
- 재물운 (wealth)
- 건강운 (health)
- 궁합 (compatibility)
- 타로 운세 (tarot)
- 바이오리듬 (biorhythm)
- MBTI 운세 (mbti)
- 월간 운세 (monthly)

#### B. 영혼을 소비하는 프리미엄 운세
고급 AI 분석이나 특별한 콘텐츠가 포함된 운세들:

**프리미엄 운세 (10-20 영혼 소비)**
- AI 사주 분석 (saju)
- AI 전통 사주 (traditional-saju)
- AI 사주 심리 분석 (saju-psychology)
- AI 토정비결 (tojeong)
- 전생 분석 (past-life)
- 운명 분석 (destiny)
- 결혼운 상세 분석 (marriage)
- 커플 매칭 분석 (couple-match)

**울트라 프리미엄 운세 (30-50 영혼 소비)**
- 창업운 종합 분석 (startup)
- 비즈니스 전략 운세 (business)
- 투자 운세 분석 (lucky-investment)
- 부동산 운세 (lucky-realestate)
- 유명인 궁합 (celebrity-match)
- 인맥 리포트 (network-report)
- 오복 운세 (five-blessings)
- 연간 종합 운세 리포트 (yearly)

### 4. 가격 정책
- 프리미엄 운세 이용권: 월 30,000원
  - 모든 프리미엄 운세 무제한 이용
  - 영혼 소비 없이 모든 콘텐츠 접근 가능
  - 특별 리포트 우선 접근권

### 5. 영혼 보상 시스템
- 일일 첫 운세: 5 영혼 보너스
- 연속 7일 접속: 20 영혼 보너스
- 친구 초대: 10 영혼 보너스
- 운세 공유: 3 영혼 보너스

### 6. UI/UX 변경사항
- 토큰 아이콘 → 영혼 아이콘 (👻 또는 ✨)
- "토큰 잔액" → "영혼 포인트"
- "토큰 구매" → "영혼 충전" (제거 예정)
- "토큰 사용" → "영혼 소비"
- 획득 시 애니메이션 효과 추가

## 구현 우선순위

1. **Phase 1**: 용어 변경
   - 모든 "토큰" 용어를 "영혼"으로 변경
   - UI 텍스트 업데이트
   - 아이콘 변경

2. **Phase 2**: 로직 변경
   - 운세 확인 시 영혼 획득 로직 구현
   - 프리미엄 운세 분류 및 소비 로직 구현
   - 데이터베이스 스키마 업데이트

3. **Phase 3**: 프리미엄 기능
   - 월간 구독 시스템 구현
   - 프리미엄 운세 콘텐츠 개발
   - 영혼 보상 시스템 구현

## 예상 효과
- 사용자 참여도 증가 (운세 확인 = 보상)
- 프리미엄 전환율 향상
- 사용자 리텐션 개선
- 게이미피케이션을 통한 재미 요소 추가
