## 1. Executive Summary

### 1.1 서비스 개요

**ZPZG (지피지기)**는 동양 전통 철학(사주, 명리학)과 현대 AI 기술을 결합한 **개인화 인사이트 플랫폼**입니다.

| 항목 | 내용 |
|------|------|
| **서비스명** | ZPZG - 지피지기 |
| **슬로건** | "AI가 분석하는 나만의 데일리 인사이트" |
| **핵심 가치** | 매일 새로운 나를 발견하는 셀프 디스커버리 |
| **플랫폼** | iOS / Android |
| **타겟 시장** | 한국 (1차), 일본/동남아 (2차) |

### 1.2 핵심 수치

| 지표 | 수치 |
|------|------|
| 인사이트 카테고리 | **35개** |
| AI 분석 기능 | 45개 Edge Functions |
| 수익 모델 | 이중 구조 (토큰 + 구독) |
| 구독 가격 | 월 ₩2,200 / 연 ₩19,000 |
