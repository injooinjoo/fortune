# iPhone 실제 기기 테스트 체크리스트 📱

## 사전 준비사항

### 1. 개발 환경 설정
- [ ] Xcode 최신 버전 설치 (16.4+)
- [ ] iPhone 개발자 모드 활성화
- [ ] Apple ID로 Xcode 로그인
- [ ] 프로비저닝 프로파일 설정

### 2. iPhone 연결 설정
- [ ] USB 케이블로 Mac과 연결
- [ ] iPhone 잠금 해제 및 "이 컴퓨터를 신뢰함" 선택
- [ ] Xcode에서 기기 인식 확인

### 3. 프로젝트 설정
- [ ] Bundle Identifier 수정 (개발용)
- [ ] Signing Team 설정
- [ ] Deployment Target 확인 (iOS 14.0+)
