# 🎨 Fortune 앱 디자인 철학

> **최종 업데이트**: 2025년 7월 26일  
> **작성자**: Fortune Design Team

## 📖 개요

Fortune 앱은 모노크롬 기반의 미니멀리즘 디자인 철학을 추구합니다. 이는 사용자가 콘텐츠에 집중할 수 있도록 하며, 깔끔하고 세련된 인터페이스를 제공합니다.

## 🎯 핵심 원칙

### 1. **Less is More (미니멀리즘)**
- 불필요한 장식적 요소 제거
- 본질적인 기능과 콘텐츠에 집중
- 여백의 미학 활용

### 2. **Monochrome First (모노크롬 우선)**
- 흑백 기반의 색상 팔레트
- 색상보다 형태, 레이아웃, 타이포그래피로 정보 전달
- 일관된 시각적 경험

### 3. **Instagram Style (친숙한 UI)**
- 사용자에게 익숙한 Instagram 스타일 채택
- 직관적인 네비게이션
- 모던하고 트렌디한 감각

### 4. **Consistency (일관성)**
- 모든 운세 타입이 동일한 디자인 언어 사용
- 통일된 컴포넌트와 패턴
- 예측 가능한 사용자 경험

### 5. **Accessibility (접근성)**
- 명확한 대비와 가독성
- 충분한 터치 영역
- 스크린 리더 지원

## 🎨 색상 시스템

### 핵심 색상
```
Primary Colors:
- Pure Black: #000000 (주요 텍스트, 아이콘)
- Dark Gray: #333333 (보조 요소)
- Instagram Black: #262626 (본문 텍스트)

Neutral Colors:
- Background: #FAFAFA (밝은 회색 배경)
- Surface: #FFFFFF (카드, 컨테이너)
- Card Background: #F6F6F6 (카드 배경)
- Text Secondary: #8E8E8E (보조 텍스트)

Accent Colors (제한적 사용):
- Instagram Orange: #F56040 (강조, CTA)
- Instagram Red: #FD1D1D (알림, 경고)
- Instagram Magenta: #E1306C (특별 강조)
```

### 색상 사용 가이드
1. **Primary Action**: 검정색 배경에 흰색 텍스트
2. **Secondary Action**: 흰색 배경에 검정색 테두리
3. **Disabled State**: 회색 톤 (#8E8E8E)
4. **Success/Error**: 시맨틱 색상 최소 사용

## 🎭 Glass Morphism

### 구현 원칙
- **투명도**: 0.1-0.2 범위
- **블러**: 10-20px
- **테두리**: rgba(255, 255, 255, 0.2)
- **그림자**: 부드럽고 미묘한 그림자

### 사용 케이스
- 플로팅 컨테이너
- 모달 배경
- 네비게이션 바
- 특별한 카드 효과

## 📐 레이아웃 원칙

### 그리드 시스템
- **모바일**: 4 컬럼 그리드
- **태블릿**: 8 컬럼 그리드
- **간격**: 16px 기본 단위

### 여백 활용
- **컨텐츠 패딩**: 24px
- **카드 간격**: 16px
- **섹션 간격**: 32px

### 정보 계층
1. **헤드라인**: 크고 굵은 텍스트
2. **본문**: 적절한 행간과 가독성
3. **보조 정보**: 작고 연한 색상

## 🎬 애니메이션 철학

### 목적성 있는 모션
- 사용자 행동에 대한 피드백
- 상태 변화의 시각화
- 콘텐츠 간 부드러운 전환

### 타이밍
- **빠른 반응**: 200ms (탭, 클릭)
- **일반 전환**: 300ms (페이지, 모달)
- **느린 전환**: 500ms (복잡한 애니메이션)

### 이징 함수
- **easeOut**: 대부분의 진입 애니메이션
- **easeInOut**: 페이지 전환
- **elasticOut**: 특별한 강조 효과

## 📱 반응형 디자인

### 브레이크포인트
- **모바일**: 360px - 767px
- **태블릿**: 768px - 1023px
- **데스크톱**: 1024px+

### 적응형 요소
- 텍스트 크기 조정
- 그리드 컬럼 변경
- 레이아웃 재구성

## 💡 디자인 결정 프로세스

1. **기능 우선**: 디자인보다 기능이 우선
2. **사용자 테스트**: 실제 사용자 피드백 반영
3. **반복 개선**: 지속적인 개선과 최적화
4. **데이터 기반**: 분석 데이터를 통한 의사결정

## 🎯 목표

Fortune 앱의 디자인은 다음을 목표로 합니다:

1. **집중**: 운세 콘텐츠에 대한 몰입
2. **신뢰**: 깔끔하고 전문적인 인상
3. **편안함**: 눈의 피로를 최소화
4. **효율성**: 빠른 정보 전달
5. **보편성**: 모든 사용자가 쉽게 사용
