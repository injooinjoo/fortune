## 🎨 디자인 원칙

### 1. **Glass Morphism**
- 모든 카드와 컨테이너에 적용
- 투명도: 0.1-0.2
- 블러: 10-20px
- 테두리: 흰색 0.2 opacity

### 2. **색상 시스템**
```
Primary: 순수 검정 (#000000)
Primary Light: 다크 그레이 (#333333)
Surface: 순수 흰색 (#FFFFFF)
Background: 밝은 회색 (#FAFAFA)
Text Primary: Instagram 블랙 (#262626)
Text Secondary: Instagram 그레이 (#8E8E8E)
Accent: Instagram 오렌지 (#F56040)
```

### 3. **애니메이션**
- 진입: fadeIn + slideY (500ms)
- 상호작용: scale 0.95 (200ms)
- 데이터: 숫자 카운팅 (1500ms)

## 💡 주요 인사이트

### 1. **정보의 계층 구조**
- 가장 중요한 정보(점수)를 크고 명확하게
- 세부 정보는 확장 가능한 카드로
- 부가 정보는 스크롤 하단에

### 2. **시각적 일관성**
- 모든 운세가 동일한 기본 구조 공유
- 모노크롬 테마로 통일된 경험
- 애니메이션 타이밍 통일

### 3. **사용자 경험**
- 즉각적인 시각적 피드백
- 직관적인 정보 구조
- 공유하기 쉬운 결과 화면

### 4. **디자인 철학**
- **Less is More**: 불필요한 장식 제거
- **모노크롬 우선**: 색상보다 형태와 구조로 정보 전달
- **Instagram 스타일**: 친숙하고 모던한 UI
