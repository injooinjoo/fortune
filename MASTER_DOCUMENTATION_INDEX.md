## 📝 업데이트 정책

### 문서 업데이트 시 규칙
1. **최종 업데이트 날짜**: 모든 문서 상단에 `최종 업데이트: YYYY년 MM월 DD일` 형식으로 기록
2. **버전 관리**: 주요 변경사항이 있을 때 버전 번호 업데이트
3. **연도 참조**: 예시나 데이터에서 연도를 사용할 때는 현재 연도 사용
4. **중복 제거**: 동일한 내용의 문서가 여러 개 있으면 하나로 통합

### 문서 카테고리
- 🟢 **활성**: 정기적으로 업데이트되고 사용되는 문서
- 🟡 **유지보수**: 가끔 참조되지만 자주 업데이트되지 않는 문서
- 🔴 **폐기 예정**: 더 이상 사용되지 않거나 중복된 문서
