## 📦 3단계: 빌드 업로드

### 🔄 방법 1: Apple Transporter 사용 (권장)
1. **Apple Transporter** 앱 다운로드:
   - Mac App Store에서 "Transporter" 검색
   - https://apps.apple.com/us/app/transporter/id1450874784

2. **IPA 파일 업로드**:
   ```bash
   # IPA 파일 경로 확인
   ls -la build/ios/ipa/

   # Transporter 앱에서:
   # 1. "+" 버튼 클릭
   # 2. build/ios/ipa/fortune.ipa 파일 선택
   # 3. "전송" 버튼 클릭
   ```

### 🔄 방법 2: 명령줄 도구 사용
```bash
# altool 사용 (API 키 필요)
xcrun altool --upload-app --type ios \
  -f build/ios/ipa/fortune.ipa \
  --apiKey YOUR_API_KEY \
  --apiIssuer YOUR_ISSUER_ID
```

### 🔄 방법 3: Xcode Organizer 사용
1. **Xcode 열기**
2. **Window** > **Organizer** 선택
3. **Archives** 탭에서 Runner.xcarchive 선택
4. **Distribute App** 클릭
5. **App Store Connect** 선택
6. **Upload** 선택 후 진행

### ⏰ 업로드 처리 시간
- 업로드 완료: 5-10분
- Apple 처리: 30분-2시간
- TestFlight에서 확인 가능
